Lección 06 - CSS Comments

¿Qué son los comentarios en CSS?

Los comentarios permiten explicar el código CSS para facilitar su comprensión y mantenimiento. Los navegadores ignoran los comentarios al mostrar la página.

Comentario en una línea

Este párrafo utiliza una clase cuyo estilo está acompañado de comentarios dentro del código CSS.

Comentario de varias líneas

También es posible escribir comentarios que ocupen varias líneas para documentar bloques completos de estilos.

Comentarios HTML y CSS

Los comentarios HTML utilizan la sintaxis <!-- comentario -->, mientras que los comentarios CSS utilizan /* comentario */.
