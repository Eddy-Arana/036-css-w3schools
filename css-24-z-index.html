Lección 24 - CSS z-index

1. z-index negativo

La imagen tiene z-index: -1, por lo que queda detrás del texto.

Este texto aparece delante de la imagen.

2. Elementos con diferentes niveles

El elemento con mayor z-index aparece encima de los demás.

Caja negra (z-index:1)
Caja gris (z-index:3)
Caja verde (z-index:2)

3. Sin utilizar z-index

Cuando no se usa z-index, los elementos se muestran según el orden en que aparecen en el HTML.

Caja negra
Caja gris
Caja verde

Resumen

  • z-index controla el orden de apilamiento.
  • Solo funciona en elementos posicionados (relative, absolute, fixed o sticky).
  • Los valores mayores aparecen delante.
  • Los valores negativos colocan elementos detrás de otros.
  • Si no se especifica un z-index, el navegador usa el orden del HTML.
