Lección 08 - CSS Backgrounds

1. background-color

Este elemento utiliza la propiedad background-color.

2. opacity

Todo el elemento, incluyendo el texto, posee transparencia.

3. Transparencia con RGBA

Solo el fondo es transparente gracias al uso de RGBA.

4. background-image

Se utiliza un gradiente como imagen de fondo.

5. background-repeat

El patrón del fondo se repite en toda el área.

6. background-repeat: no-repeat

El fondo aparece una sola vez.

7. background-position

Se establece la posición inicial del fondo.

8. background-attachment

El fondo permanece fijo mientras se desplaza la página.

9. background (Shorthand)

Este ejemplo utiliza la propiedad abreviada background.
