7. Diferencia entre Border y Outline

  • El border forma parte del tamaño del elemento.
  • El outline se dibuja fuera del borde.
  • El outline no modifica el ancho ni el alto del elemento.
  • Puede utilizarse para resaltar botones, cuadros de texto y otros elementos.
