4. Max Width

El ancho ocupa el 100% del espacio disponible, pero nunca supera los 600px.

5. Min Width

Este elemento nunca tendrá menos de 300px de ancho.

6. Min Height

Este cuadro siempre tendrá una altura mínima de 120px.

7. Max Height

Este elemento tiene una altura máxima de 120px.

Si el contenido sobrepasa ese tamaño, aparecerá una barra de desplazamiento gracias a la propiedad overflow: auto;
