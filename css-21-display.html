Resumen

  • block ocupa toda la línea.
  • inline ocupa solo el espacio necesario.
  • inline-block permite definir ancho y alto manteniéndose en línea.
  • flex organiza elementos en una fila o columna flexible.
  • grid organiza elementos en una cuadrícula.
  • display:none elimina completamente el elemento de la página.
  • visibility:hidden oculta el elemento, pero conserva su espacio.
