Lección 26 - CSS Float

float: left

La imagen flota hacia la izquierda y el texto se acomoda alrededor de ella. Este comportamiento es muy utilizado para colocar imágenes junto a un párrafo.

float: right

En este ejemplo la imagen flota hacia la derecha mientras el texto ocupa el espacio restante del lado izquierdo.

float: none

Con float: none la imagen permanece en su posición normal y el texto aparece debajo.
