Lección 13 - CSS Box Model

1. Modelo de Caja

Content

Esta área representa el contenido del elemento. Alrededor se encuentran el padding, el borde y el margen.

2. Width y Height

Width: 320px
Height: 50px
Padding: 10px
Border: 5px

Cálculo del ancho total:
320px (contenido)
+ 20px (padding izquierdo y derecho)
+ 10px (bordes izquierdo y derecho)
= 350px

Cálculo de la altura total:
50px (contenido)
+ 20px (padding superior e inferior)
+ 10px (bordes superior e inferior)
= 80px

3. Margin

El margen crea espacio alrededor del borde, separando este elemento de los demás.

4. Partes del Box Model

  • Content: Contenido del elemento.
  • Padding: Espacio entre el contenido y el borde.
  • Border: Línea que rodea el contenido y el padding.
  • Margin: Espacio exterior que separa el elemento de otros.
