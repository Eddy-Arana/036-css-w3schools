Lección 25 - CSS Overflow

overflow: visible (Predeterminado)

Este contenido es mucho más grande que el contenedor. Como el overflow está en visible, el contenido continúa mostrándose fuera del cuadro.

overflow: hidden

Este contenido se corta cuando supera el tamaño del contenedor y ya no puede verse.

overflow: scroll

Este cuadro siempre muestra barras de desplazamiento, aunque el contenido sea pequeño o grande. El usuario puede desplazarse para leer todo el texto.

overflow: auto

Las barras de desplazamiento aparecen únicamente cuando son necesarias. Este comportamiento es el más utilizado en el desarrollo web.

overflow-x y overflow-y

Este es un texto extremadamente largo que demuestra cómo overflow-x permite únicamente el desplazamiento horizontal mientras que overflow-y permanece oculto.
