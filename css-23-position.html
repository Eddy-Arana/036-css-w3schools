Diferencias entre los tipos de posición

Valor Descripción
static Posición normal del documento.
relative Se mueve respecto a su posición original.
absolute Se posiciona respecto al ancestro posicionado más cercano.
fixed Permanece fijo respecto a la ventana del navegador.
sticky Actúa como relative hasta cierto punto y luego se comporta como fixed.

Resumen

  • static Posición por defecto.
  • relative Se mueve desde su posición original.
  • absolute Sale del flujo normal y depende de un contenedor.
  • fixed Siempre permanece visible en la ventana.
  • sticky Se fija al hacer scroll.
  • Las propiedades top, bottom, left y right controlan la posición final.
