CSS Introduction

¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje utilizado para dar estilo a las páginas web. Permite cambiar colores, fuentes, tamaños, márgenes y la apariencia general de los elementos HTML.

¿Por qué utilizar CSS?

CSS separa el contenido del diseño, haciendo que las páginas sean más fáciles de mantener y modificar. Un mismo archivo CSS puede controlar la apariencia de varias páginas web al mismo tiempo.

Ventajas de CSS

Mejora la presentación de un sitio web.
Reduce la cantidad de código repetido.
Facilita el mantenimiento del diseño.
Permite adaptar la apariencia a diferentes dispositivos.
