Lección 01 - CSS Home

CSS significa Cascading Style Sheets. Se utiliza para dar estilo a las páginas web, permitiendo cambiar colores, fuentes, tamaños, márgenes y muchos otros aspectos del diseño.

En esta primera práctica se aplican estilos básicos utilizando un bloque <style> dentro del documento HTML.

Este ejemplo está basado en la lección CSS Home del tutorial de W3Schools.
