Lección 05 - How To Add CSS

1. External CSS

Este párrafo recibe su estilo desde el archivo css-05-style.css.

2. Internal CSS

Este párrafo utiliza un bloque <style> dentro del documento HTML.

3. Inline CSS

Este párrafo utiliza un estilo Inline mediante el atributo style.

Orden de Prioridad (Cascade)

Aunque esta clase tiene un color definido en el archivo CSS, el estilo Inline tiene mayor prioridad y cambia el color a morado.
