Lección 22 - CSS Max Width

La propiedad max-width establece el ancho máximo que puede tener un elemento. Si la pantalla es más pequeña, el elemento se adapta automáticamente sin provocar desplazamiento horizontal.

Ejemplo usando width

Este contenedor tiene un width:600px. Si la ventana del navegador es menor a 600px, puede aparecer una barra de desplazamiento horizontal.

Ejemplo usando max-width

Este contenedor utiliza max-width:600px. Puede reducir su tamaño automáticamente cuando la pantalla es pequeña, haciendo la página mucho más adaptable.

Contenedor Responsivo

Este contenedor utiliza max-width:900px. En pantallas grandes mantiene un ancho máximo, mientras que en pantallas pequeñas ocupa casi todo el espacio disponible.

Diferencias entre width y max-width

Propiedad Descripción
width Define un ancho fijo para el elemento.
max-width Define el ancho máximo permitido, permitiendo que el elemento se reduzca cuando sea necesario.

Resumen

  • width establece un ancho fijo.
  • max-width establece el ancho máximo permitido.
  • Es muy útil para crear diseños responsivos.
  • Evita barras de desplazamiento horizontales en pantallas pequeñas.
  • Se suele combinar con margin:auto para centrar elementos.
