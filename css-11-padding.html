Lección 11 - CSS Padding

1. Padding individual

Se utilizan las propiedades: padding-top, padding-right, padding-bottom y padding-left.

2. Padding Shorthand (4 valores)

padding: 20px 40px 60px 80px;

3. Padding Shorthand (3 valores)

padding: 20px 40px 60px;

4. Padding Shorthand (2 valores)

padding: 20px 60px;

5. Padding Shorthand (1 valor)

padding: 35px;

6. Padding con porcentaje

El espacio interno fue definido utilizando un porcentaje (8%).

Contenedor padre
