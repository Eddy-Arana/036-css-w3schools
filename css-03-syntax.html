Lección 03 - CSS Syntax

¿Qué es la sintaxis de CSS?

Una regla CSS está formada por un selector y un bloque de declaraciones. El selector indica qué elemento HTML recibirá los estilos.

Estructura básica

selector { propiedad: valor; }

Ejemplo explicado

En la regla anterior:

  • Selector: identifica el elemento HTML.
  • Propiedad: indica qué característica cambiar.
  • Valor: especifica cómo se aplicará el cambio.
  • Cada declaración termina con un punto y coma (;).
  • Las declaraciones se escriben entre llaves { }.
