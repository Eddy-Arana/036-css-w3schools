Lección 16 - CSS Fonts

Aprendiendo a utilizar fuentes en CSS.

¿Qué son las fuentes en CSS?

Las fuentes permiten cambiar la apariencia del texto de una página web.

Con CSS podemos controlar:

  • Tipo de fuente.
  • Tamaño.
  • Grosor.
  • Estilo.
  • Variantes.
Importancia: Elegir una buena fuente mejora la lectura y la apariencia profesional del sitio web.

Web Safe Fonts

Las Web Safe Fonts son fuentes que prácticamente todos los sistemas operativos tienen instaladas.

Siempre es recomendable utilizar fuentes de respaldo (Fallback Fonts).
