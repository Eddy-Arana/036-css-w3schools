Lección 19 - CSS Lists

CSS permite personalizar completamente las listas de HTML. Podemos cambiar las viñetas, los números, su posición, eliminar los marcadores y aplicar colores o estilos.

2. list-style-position

Outside (por defecto)

  • Las viñetas quedan fuera del contenido.
  • El texto inicia alineado.

Inside

  • Las viñetas forman parte del contenido.
  • El texto se desplaza junto con ellas.

Resumen

  • list-style-type: cambia el tipo de marcador.
  • list-style-position: coloca la viñeta dentro o fuera.
  • list-style-image: permite usar una imagen como marcador.
  • list-style: propiedad abreviada.
  • list-style-type: none; elimina las viñetas.
